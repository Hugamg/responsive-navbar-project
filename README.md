# Japanese Style Responsive Navbar

A clean and minimal responsive navigation bar inspired by Japanese design, built using HTML and CSS only, with a small JavaScript interaction for the mobile menu toggle. I'm not using AI for this project.

This project is intended to be reusable, customizable, and easy to integrate into any website.

--------------------------------------------------

Features

- Japanese-inspired minimalist design
- Fully responsive (desktop and mobile)
- Animated burger menu for small screens
- Clean and readable layout
- Lightweight and fast
- No CSS framework
- HTML and CSS focused with minimal JavaScript

--------------------------------------------------

Technologies Used

- HTML5
- CSS3 (Flexbox and Media Queries)
- Font Awesome (icons)
- Google Fonts (M PLUS Rounded 1c)
- Vanilla JavaScript (burger menu toggle only)

--------------------------------------------------

Responsive Behavior

Desktop (width greater than 990px)
- Full navigation with links and action buttons

Tablet and Mobile (width equal or below 990px)
- Burger menu with animated dropdown
- Navigation links and buttons displayed inside the menu

Customization

Change primary color in style.css

.action-button {
  background-color: darkred;
}

Change font  
Replace the Google Fonts link in index.html

Change background image

body {
  background-image: url("your-image.jpg");
}
